<git의 명령어>

$ git clone 웹주소 : 복제

$ git pull : 기존자료 동기화 가져오기

$ git pust : 자료 올리기

$ git status : 상태확인

$ git add 첨부자료(폴더,파일) : 첨부하기

$ git commit -m "첨부자료 설명" : 첨부자료설명 추가

$ git remove : 제거할자료

$ git reset : 초기화
